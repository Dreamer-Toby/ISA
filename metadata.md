# Project State

## Publishable After Sanitization

- `mission.md`: stable initial task and project goal.
- `context/context.md`: public machine-independent context.
- `goal.md` and `achieved/`: active and accepted goals.
- `feedback/`: Executor reports and Acceptance verdicts.
- `final-review.md`: optional first Commander-review findings.
- `final-result.md`: final whole-project verdict.
- `delivery.md`: local-only or GitHub delivery state.
- `log.md`: accepted commits and optional push history.

## Local Only

- `context/private.md` and `context/*.local.md`: machine-specific or sensitive context.
- `checkpoint/`: recovery data.
